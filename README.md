# Sandbox Utilities / サンドボックスユーティリティ

The majority of this repository consists of simple Python scripts that I created and abandoned in the past for study purposes.
I might add more later if I feel like it, or maybe I won't. Anyway, feel free to do whatever you want with it—go wild. Use it at your own risk.

このリポジトリの大半は過去に自分が勉強用に作って放置した、簡単なPythonスクリプトです。
今後の気分で増えたり増えなかったりするかもしれない。まあ煮るなり焼くなり好きに使っていいです。責任は取りません。

## Files List / ファイル一覧

- **calc.py**: Command Line Interface Calculator. Supports basic arithmetic, power, square root, and memory functions.
    - コマンドライン計算機。四則演算、累乗、平方根、メモリ機能をサポートしています。
- **fib.py**: Fibonacci Sequence Generator using caching for performance.
    - パフォーマンスのためにキャッシュを使用したフィボナッチ数列生成スクリプト。
- **gacha.py**: Gacha (Gashapon) Simulator. 
    - ガチャシミュレーター。
- **Collatz.py**: script to calculate and visualize the steps of the Collatz Conjecture for a given number.
    - 指定された数値のコラッツ予想のステップを計算して表示するスクリプト。
- **convert.py**: A tool/program to combine a silent MP4 with separate audio.
    - 音のないmp4と音声が別々であったときに合体させるやつ。
- **InfiniteSpeedtest.py**: A torture script that repeatedly runs speed tests created to verify whether unlimited mobile data is truly unlimited.
    - 携帯回線の無制限は本当なのかを検証するためにつくったスピードテストを繰り返し実行する拷問スクリプト。
- **Propaganda.py**: A censorship (propaganda) sort script I made sometime before. It's not practical.
    - 以前どっかで作った検閲（プロパガンダ）ソートのスクリプト。実用性はない。
- **qrcode.py**: QR Code Generator.
    - QRコード生成スクリプト。
- **gototravel.py**: A calculation script for the prices of the "Go To Travel" campaign that took place in Japan during the COVID-19 pandemic.
    - 過去に日本でコロナ禍のときにあった「Go To トラベル」キャンペーンの値段を出す計算スクリプト。

## Usage / 使い方

Example / 例:
```bash
python calc.py
python fib.py
```
